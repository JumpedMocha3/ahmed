Introduction to CNC Technology

  • CNC = Computer Numerical Control
  • Automated control of machine tools via computer
  • Uses G-code from CAD models

Types and Applications

  • CNC Waterjet
  • CNC Routers
  • CNC Plasma & Laser Cutters

My CNC Router Machine

  • Wood frame with 3-axis movement (X, Y, Z)
  • Adaptable platform for multiple tools
  • Linear rods and lead screws for motion

Motors and Motion Control

  • Stepper Motors for precise movement
  • Each axis driven by step/direction pulses
  • High torque at low speeds

Electronic Components

  • Arduino Uno
  • CNC Shield + Drivers
  • Power Supply

Software Workflow

  • GRBL: Open-source CNC firmware
  • CAD/CAM: Fusion 360 or Solidworks
  • G-code generation + CNCjs for control
